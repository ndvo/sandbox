Problem 01

You are to organize a party. There will be 48 toys and 42 sweets. What is the maximum number of possible guests so that each one get the same amount of each and there is no toy or sweet left?

Solution: it is the greatest common divisor. It can be found by successively dividing each number by the lowest possible value until reach one, then multiply the values that occur on both numbers.
